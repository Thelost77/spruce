# spruce v0.1.0

First public release of Spruce, a terminal music client for Jellyfin.

## Highlights

- Browse albums and tracks across Jellyfin music libraries
- Browse and queue Jellyfin playlists
- Direct audio playback through mpv
- Queue controls with shuffle and track/queue repeat modes
- Fuzzy command-palette search and in-place list filtering
- Album and track metadata editing
- Playback progress reporting to Jellyfin
- MPRIS media-key and desktop control support
- Configurable playback, keybindings, and Everforest colors
- Stable per-installation Jellyfin device identity
- Automatic migration from chapter-named playback keybindings to track names

## Security and reliability

- Owner-only config, log, and mpv IPC paths
- Owner-only temporary files for mpv authorization headers
- Isolated playback session identifiers
- Stale mpv event protection and process cleanup
- Bounded HTTP responses and request timeouts

## Installation

```sh
go install github.com/Thelost77/spruce@v0.1.0
```

## Requirements

- Go 1.26.4 or newer
- `mpv` available in `PATH`
- A Jellyfin account with a music library
- A Unix-like operating system

Linux is the primary platform. Windows is not supported in this release.
